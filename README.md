# QIDI Max 4 Firmware Config Reference

This repository exists to keep track of the QIDI Max 4 firmware configs QIDI is currently shipping.

Use tags and commit history to compare what changed between firmware releases.

## Scope

- `config/` contains the shipped Klipper, Moonraker, Fluidd, macro, box, and preset configuration files.
- Redacted hardware identifiers stay redacted.
- This is reference material for QIDI's shipped configs, not a drop-in profile for another machine.

## Firmware Checks

- GitHub Actions checks hourly for a newer QIDI Max 4 firmware release.
